Introduction

This is a basic HTML website to familiarize ourselves with it's structure. This is a basic paragraph.

Now we will use some tags:

1.strong: This is used for important text. As we can see, it increased text opacity.
2.em: This is used for emphasized text. It can be seen that it tilts the text.
3.mark: This is used for marked text. It appears as a highlight.
4.small: This is used to make small text. The text under this tag has reduced font size.
5.br: It is used to go to next line. We've already used this several times.
6.hr: Used to split the page, just as shown below.
