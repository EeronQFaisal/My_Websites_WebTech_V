Ordered Lists

Here we have a step by step list of how to build a basic HTML website:

  1. Create the HTML file:
    1. Make a new text file
    2. Rename with .html extension
    3. Open with a text editor, preferably Visual Studio Code
  2. Define the basic skeleton of the site
    1. Declare Doctype
    2. Declare html
      1. Declare head inside it
      2. Declare body
  3. Inside head:
    1. Define the title
    2. Add the meta charset with the character coding scheme
    3. Add the meta name description
  4. Inside the body, write HTML according to the website requirements

Description Lists

Here, we have a list describing the languages used in website creation:

HTML
- It is the bare bones skeleton of the website. Know as Hyper Text Markup Language.
- With HTML, we can include the basic features that make up the structure of a webpage, using various tags.
- It allows manipulation of text, symbols, even icons

Figure: HTML
CSS
- It fleshes out the webpage. Used to make it visually appealing.
- It allows the designer to manipulate colors, images, padding, etc.
Javascript
- The scripting language used in most webpages. Used to implement logic in the webpage.
- It handles control flows, loops, etc.
